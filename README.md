# Sistema de Cofrinho em Java

Este projeto foi desenvolvido como atividade prática da disciplina de Programação Orientada a Objetos (UNINTER). A proposta foi criar um sistema simples capaz de armazenar moedas de diferentes países e calcular o valor total convertido para Real.

## Estrutura do projeto
O sistema foi dividido em várias classes para praticar os conceitos básicos de orientação a objetos:

* **Moeda.java**: classe abstrata utilizada como base para as outras moedas.
* **Real.java, Dolar.java e Euro.java**: classes que herdam de Moeda e fazem a conversão de valores.
* **Cofrinho.java**: responsável por armazenar as moedas utilizando ArrayList.
* **Menu.java**: classe principal com o menu interativo no console.

## Conceitos aplicados
Neste projeto foram utilizados:

* **Herança e polimorfismo**
* **Classe abstrata**
* **Uso de ArrayList**
* **Organização do código em múltiplas classes**

## Objetivo do projeto
O objetivo principal foi praticar os conceitos iniciais de Programação Orientada a Objetos em Java e entender melhor como estruturar um sistema simples utilizando classes, herança e listas de objetos.
